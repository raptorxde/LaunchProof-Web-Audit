# LaunchProof Web Audit

Use this repository as a Claude Code skill/instruction pack for launch-readiness website audits.

When the user asks to review, QA, improve, or prepare a website project for launch:

1. Read `SKILL.md`.
2. Read `references/audit-catalog.md` when a detailed audit is needed.
3. Inspect the whole project before giving recommendations.
4. Return the report format defined in `SKILL.md`.

Do not claim an item passes unless it is verified in code, config, static files, or a live page.
