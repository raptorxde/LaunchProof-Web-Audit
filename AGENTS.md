# LaunchProof Web Audit

Use this repository as an agent instruction pack for website launch-readiness audits.

When asked to review, QA, improve, or prepare a website project for launch:

1. Read `SKILL.md` as the canonical workflow.
2. Read `references/audit-catalog.md` for detailed checks.
3. Inspect the whole project before giving recommendations.
4. Return the report format defined in `SKILL.md`.

Do not mark checklist items as passing unless they are verified in code, config, static files, or a live page.
